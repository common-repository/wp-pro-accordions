== Description ==


pro_accordions is pure HTML & CSS3 responsive accordion grid for WordPress. By this plugin you can display unlimited accordion grid at same page via short-code.


### pro_accordions by http://prowpexpert.com/



<strong>Plugin Features</strong><br />

* Fully responsive and mobile ready.
* Unlimited pro_accordions anywhere.
* Use via short-code.
* Different Theme.
* Easy input field for pro_accordions content.
* Background Image for pro_accordions area.
* Custom Active pro_accordions background color.
* Custom default background color.
* Custom font color and size for pro_accordions header.
* Custom font color and size for pro_accordions content.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>pro_accordions</strong>" activate it.<br />

After activate plugin you will see "pro_accordions" menu at left side on WordPress dashboard click "New pro_accordions" and use the options field "pro_accordions Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When pro_accordions options setup done please publish pro_accordions<br />

and then copy shortcode from top of <strong>pro_accordions Options</strong> `[pro_accordions  id="1234" ]`<br />

then paste this shortcode anywhere in your page to display pro_accordions<br />
